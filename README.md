# Wallet

## Deployment

```bash
# yarn
yarn generate

## Development Server
firebase deploy
```

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

## Development Server
Start the development server on `http://localhost:3000`:

```bash
# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# yarn
yarn build

```

Locally preview production build:

```bash
# yarn
yarn preview
```
